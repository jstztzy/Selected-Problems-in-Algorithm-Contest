### 介绍

本书是：“以一些算法题目为线索，讲解相关知识的书。”

书名暂定：算法竞赛题选 (Selected Problems in Algorithm Contest)

正文描述一道题目时，假定你所有背景知识都会，然后主要是介绍各种方法并比较。
然后所有背景知识，放在附录里去详细介绍。所选题目的准则是：
+ 没有背景知识和有背景知识时做题时间差十几倍。
+ 所用的知识或方法，出现在了很多其他题目中，具有一定的代表性。
+ 要有趣。
